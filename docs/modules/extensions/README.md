# Extensions

Modular experimental components that can be composed together.

| Extension | Description |
|-----------|-------------|
| `lattice` | Exact cross-tokenizer probability mapping via DAG |
| `translator` | Neural translator for unmatched tokens |
| `adaptive` | Speedup-aware adaptive draft length selection |
| `replay` | Replay buffer for continual distillation |
| `contrastive` | InfoNCE contrastive loss for rejection learning |
| `routing` | Dynamic multi-drafter router |
| `multitarget` | Universal drafter for multiple target models |

Each extension is developed independently and can be enabled/disabled via config.
